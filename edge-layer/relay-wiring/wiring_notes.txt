## Common Q&A

### Q: How fast is the system response?
**A:** Edge processing is near-instant (<100ms). Cloud processing adds 500ms-1s depending on network.

### Q: Can it scale to multiple cameras?
**A:** Yes! Each camera runs its own edge detection, all publish to the same MQTT broker.

### Q: What about false positives?
**A:** The cloud intelligence layer helps reduce false positives through context-aware classification and ML.

### Q: How is data secured?
**A:** MQTT supports TLS encryption. Add authentication tokens for production.

### Q: What happens if cloud connection is lost?
**A:** Edge layer continues to work independently, controlling the relay locally.

### Q: Can we add other sensors?
**A:** Absolutely! The architecture is modular - just add more publishers to MQTT.
